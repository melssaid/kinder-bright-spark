
# خطة إضافة صفحة الإعدادات وتحسين الخطوط والتصميم

بناءً على طلبك، سنقوم بإنشاء صفحة الإعدادات للمعلمات، وتحسين تصميم التطبيق ليكون أكثر جمالاً واحترافية، مع توحيد الخطوط لخط **"تجوال" (Tajawal)** للغة العربية.

## 1. صفحة إعدادات المعلمة (Settings Page)
- **مسار جديد (`/settings`)**: سيتم تصميم واجهة مقسمة إلى بطاقتين أنيقتين:
  - **المعلومات الشخصية**: نموذج لتعديل اسم المعلمة واسم الروضة/المدرسة.
  - **الأمان**: نموذج لتغيير كلمة المرور بشكل آمن.
- **تحديث قاعدة البيانات**: سيتم ربط النماذج بجدول `profiles` وحساب `auth` في Supabase لضمان حفظ التعديلات سحابياً.
- **القائمة الجانبية**: إضافة زر "الإعدادات ⚙️" في القائمة الجانبية للوصول السريع.

## 2. تحسين التصميم والخطوط
- سيتم اعتماد خط **Tajawal (تجوال)** كخط أساسي لجميع النصوص العربية، مما يعطي التطبيق طابعاً احترافياً ومقروءاً بوضوح.
- بالنسبة للأرقام والنصوص الإنجليزية، سنقوم بدمجها مع خط إنجليزي متناسق وجميل. 

### خيارات الخط الإنجليزي (المرافق لخط تجوال):
1. **Poppins (موصى به 🌟)**: خط دائري، مرن، ولطيف جداً. يتناسب تماماً مع طابع تطبيقات رياض الأطفال.
2. **Nunito**: (المستخدم حالياً) خط ناعم ومريح للعين.
3. **Inter**: خط رسمي، نظيف جداً ومستقيم (مناسب لواجهة أكثر جدية).
4. **Outfit**: خط هندسي وعصري.
